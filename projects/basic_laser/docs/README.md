# Basic Laser Project

This project contains schematics and documentation for a basic laser optical setup.

## Components

- Laser source (1064 nm)
- Focusing lens
- Screen/detector

## Schematics

- `simple_schematic.tex` - Basic laser to screen setup

## Tests

- `simple_test.tex` - Test file for verification

## Usage

To build all files in this project:

```bash
make project-basic_laser
```

To build a specific file:

```bash
# Example:
make build/basic_laser/schematics/simple_schematic.pdf
```
